    Softwares Utilizados

    Queríamos um desafio, algo que alguns de nós nunca tínhamos feito funcionar na prática. Aprendemos a desenvolver o nosso projeto durante o curso e selecionamos alguns sistemas que facilitariam nossa vida para que pudéssemos facilitar outras!

    HTML 5

    Desenvolvido em HTML 5 para facilitar a utilização em todos os dispositivos com acesso a web.

    Bootstrap 5

    Utilizamos o Framework Bootstrap 5, para uma interface visual aconchegante e bonita.

    JavaScript

    Todo cálculo desenvolvido em JavaScript para nosso projeto web ganhar vida e ajudar o usuário!

    Node js

    Fórmulas, cálculos e todo tipo de back-end desenvolvido em Node Js para produzir um site funcional e eficaz.

    Trello

    Utilizado para garantir a organização e auxiliar a Coordenação do projeto, entregando o projeto em dia.

    Visual Studio Code

    Com o melhor e mais articulado editor de texto, para desenvolver o codigo mais limpo e eficaz.
