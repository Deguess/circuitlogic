Sobre o Projeto

O intuito do projeto Simulogic é criar uma solução onde são usadas equações de eletrônica em forma de códigos em linguagem de programação que resolvam questões voltadas a circuitos elétricos e portas lógicas. A partir disso, foi criado um sistema em web que tem como função ser uma calculadora capaz de determinar valores relacionados a circuitos de meia onda, circuitos center cap, circuitos de ponte e circuitos de malha a partir de dados inseridos pelo usuário; além de ser possível resolver certos circuitos elétricos que possuem diferentes tipos de portas lógicas, também de acordo com a interação do usuário com essas portas.

  • Desenvolvido em Javascript
  • Site Responsivo utilizando CSS e Bootstrap
  • Ferramentas: Calculadora de circuitos e Simulador de portas lógicas
